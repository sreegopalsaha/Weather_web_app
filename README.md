# Simple Weather Web App

This project is a simple web app that allows users to search for weather information of any location using external APIs. It's developed as a learning project to explore API integration.

## Features
- Search for weather information of any location.
- Retrieve weather data using external APIs.
- Displays weather details on the user interface.

## Future Improvements
- Improve user interface for better usability.
- Enhance search functionality for smoother user experience.
- Explore additional APIs for more weather data features.
